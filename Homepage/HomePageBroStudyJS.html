JAVASCRIPT

What is JavaScript

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive. It plays a crucial role in enhancing user experiences by enabling features such as complex animations, clickable buttons, and popup menus.

Selecting Element Using Id Name

The most common way to select an element in JavaScript is by using its ID. You can use the document.getElementById() function to do this.

Ex. document.getElementById("demo").innerHTML = "Hello World!"

Selecting Element Using Class Name

If you want to select a class name specifically, you should put a number beside the name/string then add some method that you want to. The number 0 beside the class name means on what number of place does a specific class name you put on the element. Note: When styling, the method .style will not automatically recommend or appear in screen.

Ex.

< p class="demo" >
Hello World
< /p >
< p class="demo" >
Good Morning Sir
< /p >
document.getElementsByClass("demo")[0].style.color = "yellow";

Math Floor and Random Method

If you want to create a number guessing game or you want to make a webpage that selects a random person, you can make it happen using ".floor and .random" method.

let NumberOfStudents = 10;
//We declare a variable that equals to the equation of randomNumber
//We put the Math.random inside the Math.floor
//(Math.random()*(NumberOfStudents)+1), we multiply the number of students to Math.random
//we add +1 outside of the variable NumberOfStudents since our computer's starts the counting number from 0-1,2,3...
let randomNum = Math.floor(Math.random()*(NumberOfStudents)+1)
//print the output using the console.log
console.log( randomNum );

Switch

If you are tired to put too much if-else statements, you can use switch method. If-else statements has the same semantics to switch statement, switch has case that contains a block of code similar to if-else statements.

let x = 2;
switch(x) //The x inside of coloumn is called expression
{
case 1: console.log("x = "+x); //this first case tests the expression if the variable x is equal to 1
break; //break prevents other case to execute their own block of code
case 2: console.log("x = "+x); //this second case tests the expression if the variable x is equal to 2
break;
default: console.log("x is not equal to anything")
break;
}
